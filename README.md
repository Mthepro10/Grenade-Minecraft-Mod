# Grenade Mod
This is "Grenade Mod" : a Mod for people enjoying grenades and PVP :)

# Specifications
1) You can craft your own grenade using flint, torches and gunpowder! (use the crafting table!!!) (just like a flying TNT)

2) You can also make a apotion called _Friend of the bombs_ that will protect you from any explosions for 2 seconds. (the recipe is flint + awkward potion)
